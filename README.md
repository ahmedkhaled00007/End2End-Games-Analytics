# End2EndAnalytics_Games

*End-to-end data analytics and machine learning pipeline for video games data*, covering data cleaning, SQL analysis, Power BI dashboards, and predictive modeling.

---

## Project Overview

This project demonstrates a *complete data lifecycle* for video games data, from raw data collection to actionable insights and predictive modeling. The main goal is to analyze game sales, genres, platforms, and publishers, and build machine learning models for prediction.

Key components include:

1. *Data Collection & Cleaning:* Raw datasets collected, processed, and cleaned.  
2. *Exploratory Data Analysis (EDA):* Data trends, correlations, and insights visualized using Python.  
3. *SQL Analysis:* Data imported into SQL Server for querying, aggregation, and answering business questions.  
4. *Power BI Reporting:* Interactive dashboards showing sales, genre, publisher, and platform insights.  
5. *Machine Learning Models:* Classification and regression models built for predicting game categories and sales.  
6. *Deployment:* Models saved with pipelines for future inference.
