# Results Summary

The project produced several key insights and outcomes:

## 1. Exploratory Data Analysis
- **Top Genres:** Action, Sports, and Music games dominate sales.  
- **Top Platforms:** PS2 & DS in total sales.  
- **Publisher Insights:** Electronic Arts have multiple top-selling games across genres.  
- **Regional Trends:** Sales differ significantly across North America, Europe, and Japan.

## 2. SQL Analysis Highlights
- Top 3 most categories games ( Action-Oriented - Sports/Racing - Adventure/Story ).  
- Gold era production ( Modern Era ).  
- Most popular online games ( Duck Hunt - Wii Sports ).

## 3. Power BI Dashboards
- Interactive dashboards show sales by genre, platform, publisher, and region.  
- Dynamic filtering allows exploration of top games per year or platform.

## 4. Machine Learning Results
- **Classification Model:** Random Forest achieved strong accuracy and F1-score in predicting game categories.  

## 5. Conclusion
- The project demonstrates a **complete end-to-end pipeline**, from raw data collection to ML deployment.  
- Insights help stakeholders make data-driven decisions for game production, marketing, and sales strategies.
